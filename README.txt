# Lang Switch

## Features
- show lang at cursor when switch the lang
---
### ----------INSTALL----------

1. Clone project:
   ```bash
    git clone https://github.com/chinenar/Lang-switch.git
    cd Lang-switch
   ```
2. setup
    2.1 Move folder (My Script) to Local Disk(C:)
    2.2 Download AutoHotkey https://www.autohotkey.com/download/ahk-v2.exe
    2.3 click window + r
       2.3.1 search
   ```bash
    shell:startup
   ```
       2.3.1 Move file (ShowLang-auto start) to the folder

3. run script
   restart computer or click on file in My Script folder (ShowLang)
